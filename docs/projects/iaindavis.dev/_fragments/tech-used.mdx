## Technologies Used
- [Docusaurus][docusaurus] - React/MDX documentation framework and static-site generator from Meta
- [Storybook][storybook] - showcase/playground/test-environment for React components
- [GitHub Pages][gh-pages] - Free public hosting for static websites from GitHub
- [GitHub Actions][gh-actions] - Event based automation from GitHub
- [Vitest][vitest] - Next-gen unit testing framework built on [Vite][vite]

For a more detailed breakdown of technologies used in this project, see [the README][readme]
